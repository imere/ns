# Linux基础入门

## Linux基本命令

```bash
#切换运行级别
init [OPTION]

#显示当前目录
pwd

#复制单文件到目录
cp [OPTION]... SOURCE... DIRECTORY
#复制多文件到目录
cp [OPTION]... -t DIRECTORY SOURCE...

#删除文件/目录
rm [OPTION]... [FILE]...

#新建文件
touch [OPTION]... FILE...

#查看文件内容
cat [OPTION]... [FILE]...
```

## Windows基本命令

```cmd
@REM 显示文件/目录信息
dir

@REM 新建文件夹
md DIRECTORY
mkdir DIRECTORY

@REM 复制文件到目录
copy SOURCE DIRECTORY

@REM 永久删除文件(无提示)
del FILE

@REM 改名
rename SOURCE DIST
```
